The Ultimate Container Box

The div tag is like the blank canvas of HTML. It doesn’t add anything visual on its own — no styles, no structure — but it’s your go-to for grouping and organizing other elements. Want to wrap your header, some text, and a button together? Slap a div on it.

Block-Level by Nature

<div>is a block-level element, meaning it takes up the full width available by default and starts on a new line. Perfect for creating sections, cards, or layouts where things stack on top of each other. Think of it as building bricks in your web page’s skeleton.

Best Friend of CSS

On its own, div is kinda boring. But pair it with CSS and it becomes magical — now it can have colors, borders, spacing, animations, even act like a flex container. Whether you’re crafting complex grids or responsive designs, div is the flexible friend that adapts to your styling dreams.

The Head Element

The <head> is where you stash all the meta info, links to styles, scripts, and other behind-the-scenes magic. It doesn’t show up visually on the page, but it controls how the page behaves, loads, looks, and is understood by browsers and search engines.

    <head>
  •   <title>My Awesome Webpage</title>
    •   <!-- This sets the name of your tab on the browser. It also shows up in Google search results. -->

  •   <meta charset="UTF-8">
      <!-- This makes sure your page can show all characters like emojis, special letters, and more. -->

  •   <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <!-- This line makes your site mobile-friendly. It adjusts the layout to fit the screen size of phones, tablets, etc. -->

  •   <meta name="description" content="This is a cool site about anime and manhwa!">
      <!-- A short summary of what your site is about. Helps with SEO and shows in search results. -->

  •   <meta name="author" content="Adarsh Prasai">
      <!-- Says who made the webpage. Doesn’t change how the site looks but adds credit. -->

  •   <link rel="stylesheet" href="styles.css">
      <!-- This connects your CSS file to the page so your design and styles actually show up. -->

  •   <link rel="icon" href="favicon.ico" type="image/x-icon">
      <!-- Adds a little icon to the browser tab. Like a tiny brand logo. -->

  •   <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
      <!-- Imports a custom font (in this case, Roboto) from Google Fonts to make text look better. -->

  •   <script src="script.js" defer></script>
      <!-- This connects your JavaScript file to the page. 'defer' makes sure it loads after the page finishes loading. -->

    </head>
