CSS Float Property

The `float` property is used for positioning and formatting content, allowing text and inline elements to wrap around it. It can take values like `left`, `right`, `none`, and `inherit`.

This text wraps around the floated image. The float property allows elements to be taken out of the normal flow of the document, enabling text to flow around them.

CSS Inline-Block Property

The `inline-block` property allows elements to sit next to each other while still respecting width and height. This is useful for creating horizontal layouts without using floats.
