HTML + CSS Revision

HTML Structure

HTML stands for HyperText Markup Language. It is the skeleton of all web pages. It uses elements like <div>, <p>, <a>, and <img> to define structure and content.

  • Head: Contains meta info, title, links to CSS.
  • Body: Contains visible content of the page.

CSS Basics Recap

CSS stands for Cascading Style Sheets and styles your HTML content. It controls layout, colors, fonts, spacing, etc.

  • Selectors: Target HTML elements (e.g., #id, .class, or element names).
  • Properties: Define the styles (e.g., color, padding, margin).
  • Values: Are assigned to properties (e.g., color: red;).
